	Novamente, $0 \in a\mathbb{Z} + b\mathbb{Z}$, pois $0 = a.0 +b.0 \in a\mathbb{Z} + b\mathbb{Z}$. Tomando $x,y \in a\mathbb{Z} + b\mathbb{Z}$, assim $\exists p,q,r,s \in \mathbb{Z}; \; x=ap +bq, y=ar+bs$, então $x-y = ap +bq - (ar+bs) = a(p-r) + b(q-s) \in a\mathbb{Z} + b\mathbb{Z}$, portanto $a\mathbb{Z} + b\mathbb{Z} \subset \mathbb{Z}$ é um subgrupo.

Por definição $G/H = \{[x^{p}]: p\in \mathbb{Z}\}$, onde $[x^{p}]:=x^{p}H$ são as classes lateriais. Vamos mostrar que $G/H = <[x]>$. Com efeito, seja $\forall j \in [0, k-1]$, assim todas as potências $[x]^{j} \in G/H$ pois é um grupo, e note que $[x]^{j}=[x^{j}]$. No caso em que $j=k$ teremos $[x]^{k} = [x^{k}] = [e]$, é o elemento nulo do grupo quociente, pois $H=<x^{k}>$. Agora, se $j>k$, pelo algoritmo da divisão podemos escrever $j = mk +r, \; 0\leq r < k$ e teremos $[x]^{j}= [x]^{mk+r} = [x^{mk+r}] = [x^{r}]=[x]^{r}$, que é um caso em que $r \in [0, k-1]$, tratado anteriormente. Portanto todos os elementos do grupo quociente foram descritos como uma potência de $[x]$, isto é, $G/H = <[x]>$, é um grupo cíclico, como desejávamos.
